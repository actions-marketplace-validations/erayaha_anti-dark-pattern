# Changelog

## 1.0.0 - 2026-03-11

- Initial public release of the Anti-Dark Pattern linter.
- Added a reusable GitHub Action entrypoint for CI/CD workflows.
- Added deterministic dark-pattern detection rules for:
  - confirm shaming
  - forced continuity
  - hidden costs
  - countdown urgency
  - obstructive consent
- Added optional GitHub Models-backed analysis support.
- Added CLI output modes for text, JSON, and GitHub annotations.
