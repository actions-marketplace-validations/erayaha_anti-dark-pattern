# Anti-Dark Pattern Linter

A TypeScript-based CI/CD dark pattern linter for front-end codebases.

## Features
- Fully automated linting for dark patterns.
- Developer-first design with easy integration.
- Comprehensive test coverage.

## Installation
To install the linter, run:

```bash
yarn install
```

## Usage
You can run the linter using:

```bash
yarn lint
```

For detailed usage and options:

```bash
yarn lint --help
```
